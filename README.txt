Generated by KawaiiPrinting Discord Server Creator

QUICK START
-----------
1. Make sure Node.js v18+ is installed: https://nodejs.org
2. Open a terminal IN this folder
3. Run:  npm install
4. Run:  npm start

Your credentials are pre-filled in index.js. Keep that file private.

VOICE / MUSIC COMMANDS
----------------------
yt-dlp.exe and ffmpeg.exe are included in the bin/ folder.
They are required for music playback. Do not delete them.

ALWAYS-ON UPTIME (24/7)
-----------------------

  PM2 local:
    npm install -g pm2
    pm2 start index.js --name my-bot
